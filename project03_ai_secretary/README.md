# Project 03. 음성인식 기반 AI 개인비서 서비스 인터페이스 및 업무 자동화 흐름 구현

## 1. Project Overview

음성인식, 번역, PDF 분석, 이메일 초안 작성 및 발송 기능을 통합한 AI 개인비서 서비스 프로젝트입니다.

사용자가 음성으로 명령을 입력하고, 인식 결과를 확인·수정한 뒤, 필요한 업무를 자동화할 수 있도록 인터페이스와 기능 흐름을 구성했습니다.

## 2. Problem Definition

- 음성인식 기반 서비스는 인식 오류가 발생할 수 있습니다.
- 이메일 발송과 같은 업무 자동화 기능은 사용자의 최종 확인 단계가 필요합니다.
- 따라서 음성 명령 → 인식 결과 확인 → 수정 → 기능 실행 → 최종 확인 흐름이 중요했습니다.

## 3. Main Features

| 기능 | 설명 |
|---|---|
| 음성인식 | 사용자의 음성 명령을 텍스트로 변환 |
| 인식 결과 확인 | 변환된 텍스트를 사용자가 확인하고 수정 |
| 번역 | 이메일 또는 문장 번역 지원 |
| PDF 분석 | PDF 문서 기반 내용 분석 |
| 이메일 초안 작성 | 입력 내용을 바탕으로 이메일 초안 생성 |
| 이메일 발송 확인 | 실제 발송 전 사용자 확인 단계 제공 |

## 4. My Role

- 기본 인터페이스 구조 설계
- 팀원이 구현한 기능 코드를 서비스 화면에 통합
- 음성인식, 번역, PDF 분석, 이메일 발송 기능이 하나의 업무 흐름에서 작동하도록 화면 구성 수정
- 사용자 확인 단계와 기능 선택 흐름 개선
