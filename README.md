# DX/AI Service Improvement Portfolio

데이터 기반 문제 정의, 생성형 AI 서비스 구현, RAG 기반 학습도우미, AI 업무 자동화 프로젝트를 정리한 포트폴리오 저장소입니다.

## Portfolio Direction

**데이터 기반 DX/AI 서비스 개선 컨설팅**

공공·교육·업무 서비스 문제를 데이터로 구조화하고, 생성형 AI와 프로토타입 구현을 통해 실행 가능한 개선안을 제안하는 것을 목표로 합니다.

## Highlights

- 공공데이터 기반 주차 수요 분석을 통해 공공주차장 우선 검토 후보지를 제안했습니다.
- 생활인구, 주정차 단속현황, 공영주차장 정보, 업종별 사업체 데이터를 활용해 지역별 수요·문제 발생·공급 불균형을 분석했습니다.
- RAG 기반 AIVLE School 학습도우미에서 답변 스타일 설정, 문서 기반 예상 질문, 꼬리 질문 기능을 구현했습니다.
- 음성인식 AI 개인비서 프로젝트에서 음성인식, 번역, PDF 분석, 이메일 발송 기능이 하나의 업무 흐름으로 작동하도록 인터페이스 구조를 정리했습니다.
- 초전도체 회귀·분류 모델링 프로젝트에서 Baseline과 개선 모델을 비교하며 RMSE, MAE, R², Accuracy, Precision, Recall, F1-score, ROC-AUC 기반 성능 평가를 수행했습니다.
- 분석 결과를 표, 그래프, 발표자료, 백서 형태로 구조화하며 DX/AI 컨설팅 직무에 필요한 제안 커뮤니케이션 역량을 쌓았습니다.
