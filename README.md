# DX/AI Service Improvement Portfolio

데이터 기반 문제 정의, 생성형 AI 서비스 구현, RAG 기반 학습도우미, AI 업무 자동화 프로젝트를 정리한 포트폴리오 저장소입니다.

## Portfolio Direction

**데이터 기반 DX/AI 서비스 개선 컨설팅**

공공·교육·업무 서비스 문제를 데이터로 구조화하고, 생성형 AI와 프로토타입 구현을 통해 실행 가능한 개선안을 제안하는 것을 목표로 합니다.
