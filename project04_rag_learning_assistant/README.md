# Project 04. RAG 기반 AIVLE School 학습도우미 구축 및 사용자 질문 흐름 개선

## 1. Project Overview

AIVLE School 관련 문서를 기반으로 사용자의 질문에 답변하는 **RAG 기반 학습도우미 서비스**를 구축한 프로젝트입니다.

단순 챗봇이 아니라, 문서 기반 검색과 생성형 AI를 결합하여 사용자가 교육 과정, 학습 전략, 프로젝트, 취업 준비 관련 정보를 더 쉽게 탐색할 수 있도록 설계했습니다.

특히 사용자가 한 번의 질문으로 끝내지 않고 학습 흐름을 이어갈 수 있도록 **답변 스타일 설정**, **문서 기반 예상 질문**, **꼬리 질문 추천 기능**을 구현했습니다.

---

## 2. Problem Definition

일반 생성형 AI는 사용자의 질문에 자연스럽게 답변할 수 있지만, 특정 문서나 교육 자료에 근거한 답변이 필요한 상황에서는 다음 문제가 발생할 수 있습니다.

- 문서에 없는 내용을 그럴듯하게 답변할 가능성
- 사용자가 어떤 질문을 해야 할지 모르는 문제
- 답변 이후 추가로 무엇을 확인해야 할지 알기 어려운 문제
- 긴 문서에서 필요한 정보를 직접 찾기 어려운 문제

따라서 본 프로젝트에서는 **AIVLE School 백서와 관련 자료를 기반으로 답변하는 RAG 구조**를 활용해, 사용자가 문서 기반 정보를 더 쉽게 탐색할 수 있는 학습도우미를 구현했습니다.

---

## 3. System Flow

```text
사용자 질문 입력
→ 문서 검색
→ 관련 문단 추출
→ LLM 답변 생성
→ 답변 스타일 적용
→ 문서 기반 예상 질문 생성
→ 꼬리 질문 추천
→ 사용자 후속 질문 유도
```

이 흐름을 통해 단순 질의응답이 아니라, 사용자가 문서를 바탕으로 계속 학습을 이어갈 수 있는 구조를 만들고자 했습니다.

---

## 4. Main Features

| 기능 | 설명 |
|---|---|
| 문서 기반 질의응답 | 업로드된 문서 내용을 기반으로 사용자의 질문에 답변 |
| 답변 스타일 설정 | 사용자의 목적에 따라 답변 방식 선택 가능 |
| 예상 질문 생성 | 문서 내용을 바탕으로 사용자가 물어볼 만한 질문 추천 |
| 꼬리 질문 추천 | 답변 이후 이어질 수 있는 후속 질문 제공 |
| Streamlit UI | 사용자가 쉽게 접근할 수 있는 웹 기반 인터페이스 구성 |
| 백서 기반 정보 탐색 | AIVLE School 백서 내용을 기반으로 학습·지원·취업 관련 정보 탐색 지원 |

## 8. Key Results

- AIVLE School 백서를 기반으로 사용자가 교육 과정, 학습 전략, 프로젝트, 취업 준비 정보를 질문할 수 있는 학습도우미를 구현했습니다.
- 답변 스타일 설정 기능을 통해 사용자가 원하는 방식으로 답변을 받을 수 있도록 구성했습니다.
- 예상 질문과 꼬리 질문 기능을 통해 사용자가 다음 질문을 쉽게 이어갈 수 있도록 했습니다.
- 단순 챗봇이 아니라, 문서 기반 정보 탐색과 학습 흐름을 지원하는 인터페이스를 설계했습니다.

---

## 9. Key Learning

이 프로젝트를 통해 RAG 기반 서비스는 단순히 LLM을 호출하는 구조가 아니라, **문서 검색 → 관련 근거 추출 → 답변 생성 → 후속 질문 유도**까지 연결되어야 사용성이 높아진다는 점을 배웠습니다.

또한 AI 서비스에서 중요한 것은 답변을 생성하는 기능 자체뿐 아니라, 사용자가 어떤 질문을 해야 할지 모르는 상황까지 고려해 **질문 흐름을 설계하는 것**이라는 점을 경험했습니다.

---

## 10. Limitations

- 답변 정확도에 대한 정량 평가는 수행하지 않았습니다.
- 따라서 본 프로젝트에서는 “정확도 향상”이 아니라 **문서 기반 답변 신뢰도 향상을 목표로 설계**했다고 표현합니다.
- 실제 운영 서비스 수준의 사용자 로그 분석, 피드백 수집, 성능 모니터링은 수행하지 않았습니다.
- 문서 품질과 검색 품질에 따라 답변 품질이 달라질 수 있습니다.

---

## 11. Improvement Plan

향후 개선한다면 다음 기능을 보완할 수 있습니다.

- 답변에 사용된 문서 출처 표시
- 검색된 근거 문단 함께 제공
- 질문 유형별 프롬프트 분기
- 사용자 피드백 기반 답변 개선
- 답변 정확도 평가 기준 설계
- 관리자용 문서 업로드 및 관리 기능 강화
- 사용자 질문 로그 기반 FAQ 자동 생성
