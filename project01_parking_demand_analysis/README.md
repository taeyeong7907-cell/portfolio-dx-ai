# Project 01. 공공데이터 기반 주차 수요 분석 및 공공주차장 우선 검토 후보지 제안

## 1. Project Overview

지자체의 공공 주차시설 확충 의사결정을 지원하기 위해 생활인구, 주정차 단속현황, 공영주차장 정보, 업종별 사업체 데이터를 분석한 프로젝트입니다.

단순히 단속건수가 많은 지역을 선택하는 것이 아니라, 수요·문제 발생·공급 수준을 함께 고려하여 공공주차장 설치 우선 검토 후보지를 제안하는 것을 목표로 했습니다.

## 2. Problem Definition

- 공공 주차시설 확충은 예산과 공간 제약이 크기 때문에 데이터 기반의 우선순위 판단이 필요합니다.
- 주차 문제는 단순한 주차면수 부족이 아니라 생활인구, 시간대별 수요, 단속 발생, 기존 주차 인프라의 불균형과 연결됩니다.
- 따라서 수요와 공급의 미스매치를 분석해 우선 검토 후보지를 도출했습니다.

## 3. Data Used

| 데이터 | 활용 목적 |
|---|---|
| 생활인구 데이터 | 지역별·시간대별 잠재 주차 수요 파악 |
| 주정차 위반 단속현황 | 실제 주차 문제 발생 정도 확인 |
| 공영주차장 정보 | 기존 공공 주차 인프라 공급 수준 확인 |
| 업종별 사업체 데이터 | 생활권 및 방문 수요 변화 해석 |

## 4. Analysis Process

1. 행정동 단위 데이터 정리
2. 생활인구 시간대·연령대 패턴 분석
3. 주정차 단속건수 비교
4. 공영주차장 공급 수준 비교
5. 업종별 사업체 변화 확인
6. 히트맵과 대시보드 기반 후보지 검토

## 5. Key Results

- 신정동은 생활인구 규모와 주정차 단속건수가 모두 높은 지역으로 확인되었습니다.
- 신정3동과 신정4동을 중심으로 생활인구 수요가 집중되는 패턴이 나타났습니다.
- 잠원동은 신정동보다 전체 규모는 작지만 특정 시간대·연령대 중심의 수요 집중 가능성이 확인되었습니다.
- 최종적으로 신정동과 잠원동을 공공주차장 설치 우선 검토 후보지로 제안했습니다.

## 6. My Role

- 데이터 분석 결과 정리
- 표 및 그래프 구성
- 후보지 판단 기준 정리
- 발표 자료 구성 보조
- 분석 결과를 정책 제안 관점으로 구조화

## 8. Limitations

- 실제 부지 확보 가능성, 예산, 토지 소유, 법규 검토는 수행하지 않았습니다.
- 따라서 본 프로젝트는 “최적 입지 선정”이 아니라 “우선 검토 후보지 제안”으로 표현합니다.

## 9. Improvement Plan

- 토지 이용계획 및 공공부지 데이터 결합
- 후보지별 정량 점수화 기준 보완
